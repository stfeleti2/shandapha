# next-app-router-starter

Reference notes for the `next-app-router` generator recipe.

Use this example to validate:

- `ThemeProvider` wrapping in the root layout
- generated `tokens.json` and `theme.css`
- template-first route generation
- reversible patch-install diffs for existing Next.js apps

Expected output from the generator includes a working App Router shell, an
example verification route, and doctor checks for provider wiring and theme
assets.
