# team-roles-starter

Membership management starter with invites, roles, seats, and audit framing.

## Install metadata
- Shell: AdminShell
- Group: saas
- Required packages: none
- Required routes: generator default
- Theme dependencies: theme.css, tokens.json

## Slots
- memberTable
- invitePanel
- seatSummary

## Outputs
- member roster
- pending invites

## Framework support
- react-vite: supported
- next-app-router: supported
- wc-universal: experimental
- blazor-wc: experimental
