# dashboard-home

Overview shell with KPI rails, chart cards, activity cards, and action slots.

## Install metadata
- Shell: AdminShell
- Group: saas
- Required packages: none
- Required routes: generator default
- Theme dependencies: theme.css, tokens.json

## Slots
- heroActions
- kpiRail
- chartRegion
- tableRegion

## Outputs
- dashboard overview
- trend summary
- team activity

## Framework support
- react-vite: supported
- next-app-router: supported
- wc-universal: experimental
- blazor-wc: experimental
