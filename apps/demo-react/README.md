# react-vite-starter

Reference notes for the `react-vite` starter recipe.

This example should demonstrate the lowest-friction React integration:

- provider setup in `main.tsx`
- token and theme asset loading
- template mounting without App Router assumptions
- optional patch-install guidance for existing Vite projects

Use it when validating that the generator stays framework-aware without forcing
monorepo or Next.js-specific structure.
