---
title: Overview
description: Shandapha is a UI modernization and continuity platform.
---

Shandapha helps teams modernize real existing apps without rewriting the stack.

The core flow is:

1. Inspect the existing app.
2. Map the brand and semantic tokens.
3. Install premium product surfaces with merge-safe patches.
4. Verify the result with Doctor.
5. Govern continuity across teams and products.
